# DAFTech Driver License System

## Overview
A Flutter mobile application and .NET backend system for driver license registration and verification.

## Features
- User authentication with JWT
- Driver registration with QR code scanning
- License verification (Real/Fake detection)
- OCR text extraction from licenses
- Duplicate license detection

## Tech Stack
- Backend: .NET 10 Web API
- Database: SQL Server
- Mobile: Flutter
- Authentication: JWT tokens

## Project Structure
- DAFTech.DriverLicenseSystem.Api/ - Backend API
- mobile_flutter/ - Flutter mobile app

## Setup Instructions
1. Clone the repository
2. See SETUP_GUIDE.md for detailed setup
3. Run backend: dotnet run
4. Run mobile: flutter run
